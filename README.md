# Installation

Clone this repository.
```sh
git clone git@github.com:andriansyahnc/nc-typescript-nodejs.git
cd nc-typescript-nodejs
```

Install the dependencies
```sh
npm install
```

Run the application in development
```sh
npm run start:dev
```

Build and Run the application
```sh
npm run start
```

Run unit tests
```sh
npm test
```
